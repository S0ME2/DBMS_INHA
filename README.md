# Welcome to the Database Course Preparation!

This repository contains all the lecture notes you'll need to help you prepare for Database course in INHA university in Tashkent. Inside, you'll find Markdown files for each key topic, packed with explanations, examples and diagrams (if I am in good mood) to help you understand the concepts.

Whether you're reviewing for an exam, working on assignments or just getting a better grasp of databases, these notes are here to support you through it.

Feel free to explore, refer back to the notes whenever needed, and don't hesitate to contribute if you have any suggestions or additions!

Happy learning!
